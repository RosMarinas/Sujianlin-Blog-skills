## 适用问题

旨在结合局部高阶精度与全局稳定性，求解兼具尖锐关键点高阶吻合与大范围平滑趋势极低误差的函数逼近。常用于替换神经网络底层实现中计算复杂、费时的特殊函数。

## 核心变换

将“单一的解析展开极易在全局发散（泰勒展开的弱点）”与“单一的数值优化易导致关键点曲率不平滑”的矛盾，变更为对不同特性的复合拼装。

## 典型步骤

1. **目标分解**：将待逼近目标分解为需要精密把控的主导局部（例如 $x=0$ 附近的奇点、极值点）与宽广的全局范围。
2. **泰勒提取局部结构**：在主导局部，通过泰勒级数精确匹配导数，提取出低阶的多项式作为局部底座，确保在此区域曲率严丝合缝。
3. **全局残差寻优**：剔除局部拟合已经覆盖的趋势，剩余的残差在一个非常宽的截断区间内利用最优化手段寻找修正多项式（通常以 Min-Max 即无穷范数最小化为目标）。
4. **加权并联**：将两个尺度的近似函数按区域平滑拼接为混合替代公式。

## 直觉

想临摹一座连绵且中间有一座高峰的山脉。如果你仅仅关注高峰怎么起伏（泰勒展开），你会把远处的平原画得高耸入云；如果你只关心整体画面比例匀称（单纯曲线全局拟合），高峰由于太尖锐反而会被磨平成为钝角。混合拟合的直觉是，在山峰处我们拿放大镜一笔一划按照几何规律画出来，至于远处的原野我们只用几根宽大的线条粗粗填满它，只要保证最大偏差不刺眼就好。

## 边界

- 拼装后的初等函数不一定具备高阶的可微性，若需要被逼近的函数用作含有高阶导数惩罚的网络激活层，可能会因为微小间断点导致训练受阻。
- 在构建混合函数时，全局误差的控制区域如果取得过大（如取到无穷远处），会导致修正项本身不可避免地发生震荡。

## 例子

GELU 的初等函数近似。
目标是逼近 $\Phi(x)$，高斯累积分布函数在计算设备上十分昂贵。单纯的泰勒展开在远离 0 的地方会爆掉。
GELU 的作者使用混合技巧，即在 0 附近逼近其导数形式，同时通过数值求解使得逼近的反正切 / 双曲正切形状在全实数域内的最大绝对误差最小。最终将其合成为一条包含 $0.044715 x^3$ 的初等多项式修正形式：$x \cdot \sigma(1.702x)$ 或 $0.5x(1 + \tanh(\sqrt{2/\pi}(x + 0.044715x^3)))$，它完美保留了 0 附近的细微起伏，而在两端又极其紧密地贴合 0 和 $x$ 这两条渐近线。

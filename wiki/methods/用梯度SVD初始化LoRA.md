# 用梯度SVD初始化LoRA

## 适用问题

高维对象可以用低维结构近似时，特别是在微调大模型时希望低秩适配器（LoRA）从一开始就获得尽可能接近全参数微调下降方向的起步点。

## 核心变换

将完全零初始化或正态分布初始化的瞎猜，替换为对“模型在当前初始任务上最渴望改变的方向（即初始梯度）”的主成分截取。

## 典型步骤

1. 选取一批代表性样本，计算初始权重矩阵的完整梯度 $G_0 = \nabla_{W_0}\mathcal{L}$。
2. 对该初始梯度进行奇异值分解 $G_0 = U\Sigma V$。
3. 取左奇异向量矩阵 $U$ 的前 $r$ 列初始化微调矩阵 $A$。
4. 取右奇异向量矩阵 $V$ 的前 $r$ 对应行（或论文中由于对称分配而取 $r+1 \sim 2r$ 行）初始化微调矩阵 $B$。

## 直觉

普通的 LoRA 保证了“第 0 步不改变原模型”，但无法保证“第 1 步走向正确的方向”。如果我们计算出全量微调在第 1 步应该向哪走，然后把这个完整步幅方向做一个 SVD（最佳低秩近似），拆给 A 和 B，这样 LoRA 的第一步更新就能紧紧贴合全量微调的更新，起步就领跑。

## 边界

初始化阶段需要计算一次全尺寸的梯度并进行 SVD，这在显存严重不足的设备上极难操作（只能采取按参数串行计算等显存换时间的策略）；依赖一小批样本给出的“初始梯度”，如果样本不具代表性或存在离群点，初始化的指导意义会大大减弱；对 SGD 推导完美但对 Adam 存在符号衰减的理论缺口。

## 例子

LoRA-GA（Gradient Approximation）正是应用此法，先采样数据算一遍大矩阵的梯度，分解后把最大奇异值对应的子空间分配给 A 和 B，大幅改善了模型在 GLUE 等基准上的少步数训练效果和极限准确率。

## 证据

- ev::10226::LoRA-GA 提出为了让 LoRA 的初始更新步尽可能对齐全量微调的更新结果，将问题的零阶误差最小化转化为对初始梯度的低秩近似，推导出了通过对初始梯度进行 SVD 来初始化 $A$ 和 $B$ 的策略。

# 矩阵分块递归求逆

## 适用问题

适用于 DeltaNet/线性 Attention 中形如 Lambda + QK^T odot M^- 的对角+低秩下三角矩阵求逆。

## 核心变换

下三角矩阵求逆 -> 分块递归 -> 低维缓存变量更新。

## 典型步骤

1. 把目标矩阵按下三角块矩阵逆公式递归拆分。
2. 利用 T[l:l+1,:l]=Q[l:l+1]K[:l]^T 暴露低秩结构。
3. 缓存 K[:l]^T T[:l,:l]^{-1}，避免每步 l x l 乘法。
4. 按单步或 chunk 更新逆矩阵块及缓存变量。

## 直觉

朴素递归的瓶颈是历史块乘法；低秩结构说明历史影响可由 K^T T^{-1} 这类低维缓存生成。

## 边界

需要下三角 Mask 和低秩因子结构；普通稠密矩阵不能直接使用该复杂度结论。

## 例子

- 11072 对 T=Lambda+QK^T odot M^- 推导分块逆，并指出缓存 K[:l]^T T[:l,:l]^{-1} 后每步不再涉及 O(l^2) 运算。

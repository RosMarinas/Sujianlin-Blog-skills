# 基于生成函数的欧拉乘积展开

## 适用问题

当需要研究自然数域上的求和级数，并且级数项能够映射到自然数的素数因子分解时，可以使用该方法。典型问题如黎曼 $\zeta$ 函数的改写、素数倒数之和发散的证明、任意两个自然数互质概率的推导。

## 核心变换

输入是一个以全体自然数为索引的加法级数（例如黎曼 $\zeta$ 函数），输出是仅以素数为独立变量的无穷乘积结构。这种变换将极其复杂的“累加”转变成了互相正交的“独立乘法”。

以源文 `1505` 为例：

```tex
\xi (s)=1+\frac{1}{2^s}+\frac{1}{3^s}+\frac{1}{4^s}+\dots
```

将其乘以 $1/2^s$ 并相减后，消去了所有分母含偶数的项。反复利用算术基本定理（类似埃拉托色尼筛法），经过对所有素数进行筛选，最终能提取出纯粹的素数表示：

```tex
\xi (s)=\prod_{p} (1-p^{-s})^{-1}
```

## 典型步骤

1. 写出待研究的加法级数。
2. 选定最小素数（如 $p=2$），在原级数中提取对应因子并将其从整个级数中减去，筛掉其所有倍数。
3. 类似埃拉托色尼筛法，对剩余级数依次乘以下一个素数（$p=3,5,7,\dots$）的相关因子并减去，直到把所有合数项完全删去。
4. 由于最后只剩下 $1$，将左侧所有的 $(1-p^{-s})$ 移项到右侧，形成倒数形式的无穷乘积。
5. 通过这种正交的乘法结构，独立研究每个素数对应的解析行为。

## 直觉

自然数的构成像是一个混合在一起的交响乐团，每个自然数是素数的组合，如果我们直接计算它们的加法和，就像是在听和声的叠加，很难分析规律。而由于算术基本定理的唯一分解性，所有的自然数其实是由不同素数作为独立基底“乘”出来的。欧拉乘积展开就像是对这首音乐做了傅里叶变换，把混杂的频率拆解成了独立的正弦波（素数）。

## 边界

- 只有依赖算术基本定理（即唯一分解存在且绝对）的求和级数才可能适用。
- 要求在极限情况下级数部分收敛（如 $s>1$ 时黎曼 $\zeta$ 才能严格保证两端收敛及等式成立）。
- 不能盲目套用于非积性函数的求和操作。

## 例子

- `1505` 黎曼 $\zeta$ 函数：把 $1/n^s$ 的求和转化为素数倒数的乘积。
- `1484` 素数无穷证明：借用欧拉乘积展开反证素数如果有限则会导致不合理的调和级数收敛。

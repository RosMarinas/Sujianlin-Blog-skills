# 高阶摄动级数逐阶消零法

## 适用问题

适用于高斯型微扰积分需要比一阶封闭近似更多有效位、且 epsilon/a^2 较小的渐近计算。

## 核心变换

单参数 A 近似 -> A 的 epsilon 级数 -> 逐阶积分消零递推。

## 典型步骤

1. 将 A 写成 a+a1 epsilon+a2 epsilon^2+...。
2. 比较原积分与 exp(-A x^2) 近似积分的差。
3. 按 epsilon 展开被积函数并逐项积分。
4. 令每阶系数为 0，解出 a1、a2、a3 等递推系数。

## 直觉

一个常数 A 只能消掉一阶误差；把 A 本身展开成级数后，每个新系数都可用来抵消更高一阶误差。

## 边界

3241 明确说明该级数是渐近表达；epsilon/a^2 小时截断前几项有用，a -> 0 或 epsilon -> infinity 时会失效。

## 例子

- 3241 令 A=a+a1 epsilon+a2 epsilon^2+...，逐项积分后令各阶为 0，得到 a1=3/(2a)、a2=-39/(8a^3)、a3=657/(16a^5) 等。

# 指数变系数展开法

## 适用问题

适用于高斯背景加非线性扰动的积分近似，尤其是直接泰勒展开在强扰动或 a 接近 0 时表现不稳的场景。

## 核心变换

固定高斯系数展开 -> 可调 A 吸收扰动 -> 一阶修正消零的封闭近似。

## 典型步骤

1. 把 e^{-a x^2-epsilon x^4} 改写为 e^{-A x^2} 乘以剩余扰动。
2. 对剩余扰动作低阶展开。
3. 计算一阶修正项的积分。
4. 选择 A 使该修正项为 0，并用 sqrt(pi/A) 作为封闭近似。

## 直觉

不把所有扰动暴露在多项式级数里，而是把一部分重新吸收到指数主项中，可保留高斯积分的可积结构。

## 边界

当前页面只保留一阶封闭近似证据；高阶系统推广见“高阶摄动级数逐阶消零法”。

## 例子

- 3217 引入 A，令 2A(a-A)+3epsilon=0，得到 A=(a+sqrt(a^2+6epsilon))/2，并用 sqrt(pi/A) 近似原积分。

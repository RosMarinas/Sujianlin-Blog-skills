# 对比学习梯度累加等效变换

## 适用问题

适用于显存不足以容纳大 batch 对比学习，但希望保留大 batch 内负样本梯度效果的句向量或表征学习训练。

## 核心变换

全 batch 对比梯度 -> stop_gradient 等效内积 -> 小 batch 梯度累积。

## 典型步骤

1. 推导对比学习 softmax 损失对相似度矩阵的梯度。
2. 用 stop_gradient 固定概率权重，得到梯度等效的线性相似度损失。
3. 把相似度内积拆为 h_i 与停止梯度的 h_j 项。
4. 先计算并缓存辅助向量，再按小 batch 对每个 h_i 回传梯度。

## 直觉

普通梯度累积不等价，因为 softmax 分母跨样本耦合；显式算出耦合项并将其作为常数辅助向量，就能把回传拆回单样本。

## 边界

推导依赖相似度为向量内积形式；BN 等跨样本层仍可能破坏等效性。

## 例子

- 8471 推导用 p_ij^(sg)-t_ij 和 stop_gradient 内积替代原相似度损失，可得到相同梯度并分两阶段计算。

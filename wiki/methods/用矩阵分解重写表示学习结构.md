# 用矩阵分解重写表示学习结构

## 适用问题

当一个表示学习模型看上去是神经网络、Embedding 层、词向量训练或压缩编码，但其核心结构可以写成“大矩阵 -> 低维中间表示 -> 重建大矩阵”时，可以使用这个方法。它用于判断模型是否本质上在做矩阵分解、低维压缩或聚类结构暴露。

输入对象通常是高维矩阵、one-hot 输入、词表共现矩阵、线性自编码器的编码/解码权重，或 Word2Vec/CBOW 中的词向量表和输出层。

## 核心变换

核心变换是把一个表示学习结构重写成矩阵乘法因子：

```math
M_{m\times n}\approx A_{m\times k}B_{k\times n}
```

或把自编码器写成编码矩阵与解码矩阵的组合。这样，模型输出不再只被看成神经网络前向传播，而是被看成对原始高维关系矩阵的低维重建。输出对象可以是矩阵分解公式、低维表示、聚类解释，或对 Word2Vec 与 SVD 结构等价性的判断。

## 典型步骤

1. 把原始数据或模型输入写成矩阵，例如文档-词矩阵、词上下文分布矩阵或 one-hot 输入矩阵。
2. 找到模型中的低维瓶颈层，明确它对应的矩阵因子维度。
3. 将前向传播中的线性层写成矩阵乘法，得到压缩再重建的结构。
4. 如果讨论 SVD 的语义，将矩阵的行和列分别解释为两个聚类空间，中间矩阵解释为类间匹配。
5. 对 Word2Vec/CBOW 这类模型，把词向量求和和全连接输出层还原为一个三层网络结构。
6. 最后单独检查非线性激活、softmax、预测目标和训练技巧，因为这些是与纯线性 SVD 视角不同的部分。

## 直觉

矩阵分解把一个巨大的关系表压缩为少数低维因子；线性自编码器也把输入压到低维瓶颈再重建。源文的关键直觉是：这两件事在结构上是同一种操作。SVD 不只是降维或省存储，它还可以被理解为发现行对象、列对象及其匹配关系的聚类结构，因此可以给一些神经表示学习模型提供可解释的几何和概率视角。

## 边界

这个方法适合结构解释，不等于证明所有模型训练目标完全等价。源文在 Word2Vec 例子中明确保留差异：Word2Vec 通过上下文预测当前词，输出端通常接 softmax，训练目标和优化技巧也不同；线性自编码器与 SVD 的对应也要求暂时忽略激活函数。若模型的关键收益来自非线性、注意力、采样目标或复杂训练策略，不能只用矩阵分解解释全部行为。

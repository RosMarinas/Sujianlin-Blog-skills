# 用互信息发现词语边界

## 适用问题

当面对一种完全陌生的语言符号序列（或者需要发现已知语言中未被收录的新组合模式），并且缺乏现成的分词字典时，可利用该方法来发现哪些符号应该被当作一个整体（词语）。典型的适用场景包括中文新词发现、无监督分词词库构建、以及特定垂直领域的特征短语提取。

## 核心变换

输入是原始的字符或标记序列及其在全语料中的统计频次，输出是经过判定后的组块边界。

此变换的核心是用信息论指标来近似衡量两个符号结合的“紧密程度”。依据最小熵原理的推导，两个相邻元素 $(a, b)$ 合并能有效降低平均字信息熵的一个充分条件是它们满足一定的共现特异性，而该特异性的近似下界即为点互信息（PMI）：

```tex
F_{ab}^* = P(a,b) \left(\ln \frac{P(a,b)}{P(a)P(b)} - 1\right) \gg 0
```

这就自然要求 $\ln \frac{P(a,b)}{P(a)P(b)} > 1$，将合并问题化简成了局部的互信息阈值切分问题。

## 典型步骤

1. **统计基础频次**：从大量的语料中统计每个基本单元（如单字）的概率 $P(a), P(b)$，以及相邻两字共现的概率 $P(a,b)$。
2. **计算局部互信息**：对任意相邻对 $(a,b)$ 计算点互信息 $PMI = \ln \frac{P(a,b)}{P(a)P(b)}$。
3. **当机立断（切分）**：分别设定共现频率阈值和互信息阈值，当序列中某相邻对的互信息低于该阈值时，直接在此处将序列切开。
4. **频率截断**：统计切分出的所有“准词语”候选的频次，过滤掉低频片段。
5. **去冗（可选）**：通过检查更长的组合词内部是否可以被子词以更大乘积概率解释，剔除无独立成词必要的“虚假词”（如“的主要”）。

## 直觉

词语的本质是“经常组合出现、且不仅是因为各自频率都很高才偶尔碰到一起”的固定套路。点互信息的分子代表它们实际碰在一起的概率，分母代表如果它们各自随机游走时碰在一起的概率。如果分子远大于分母（互信息很大），说明它们之间有一种强烈的内在吸引力，此时它们就不应该被切开，而应该作为一个整体（词）被记住。反之，如果吸引力不足，就“当机立断”将其切开。

## 边界

- **只看相邻二元对**：这种简化的 PMI 切分只考察了相邻的 $2-gram$ 统计，可能会在某些三元强关联但二元较弱的情况下失误（如姓名）。
- **短词偏好与长词稀疏**：长词的频次统计通常不充分，容易因频率估计偏高而被保留，需要辅助以基于长度的降权或“去冗”操作。
- **阈值敏感**：互信息阈值的选取对于最终词库的粒度影响极大，需要根据人工经验微调。

## 例子

- `5476` 基于互信息的无监督词库构建：在构建微信公众号语料的词库时，通过设置 PMI 阈值切分出准词，再用子词概率乘积筛去类似“的主要”这样无意义的长词组合。

- `ev::5476::互信息计算`：基于降低语料平均记忆信息熵的目标，推导出近似的下界公式，证明 $PMI > 1$ 是合理合并的前提。
- `ev::5476::阈值切分`：逆向运用互信息公式，对小于阈值的相邻字“当机立断”进行切分。
- `ev::5476::去冗过滤`：利用一元分词的概率连乘逻辑，将“子词概率乘积大于整体概率”的长词从词典中剔除。

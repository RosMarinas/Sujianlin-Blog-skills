# 句向量对比学习分类训练

## 适用问题

适用于有相似句对或可用 dropout 构造正例视图的句向量对比学习。

## 核心变换

句向量矩阵 -> batch 内相似度分类 logits -> 对比学习损失。

## 典型步骤

1. 编码 batch 中的句子得到句向量矩阵。
2. 对句向量做归一化并计算两两内积或余弦相似度。
3. 屏蔽自身或按增强视图确定正例位置。
4. 对每一行做 softmax 分类损失，提升正例相似度、压低 batch 内负例。

## 直觉

无需显式构造全量负样本库，当前 batch 中的其他句子就能提供负例近似；SimCSE 进一步用两次 dropout 前向构造正例。

## 边界

batch 内负样本质量和 batch 大小会影响训练；该页不处理跨 batch 队列或硬负样本挖掘。

## 例子

- 7427 用 batch 内 CLS 向量两两内积做分类训练；8348 将两次 dropout 得到的同一句向量视为正例。

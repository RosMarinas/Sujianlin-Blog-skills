# 用切空间投影改写约束最速下降

## 适用问题

当优化器不仅要让损失下降，还必须让参数留在某个等式约束定义的可行域中时，可以使用这个方法。典型问题是：更新前后参数都应位于单位球面、正交约束、Stiefel 流形或谱约束面上，此时“梯度反方向”不一定是可行方向。

## 核心变换

这个方法的输入是带约束的最速下降目标，输出是满足约束的一阶可行下降方向。以 `11196` 的超球面例子为基本证据，源文先把最小作用量原则写成步长约束下的下降目标，再要求更新后的参数仍在单位球面上：

```tex
\max_{\boldsymbol{\varphi}} \langle\boldsymbol{g},\boldsymbol{\varphi}\rangle
\quad \text{s.t.}\quad
\Vert\boldsymbol{\varphi}\Vert_2=1,\,
\Vert\boldsymbol{w}-\eta\boldsymbol{\varphi}\Vert_2=1,\,
\Vert\boldsymbol{w}\Vert_2=1
```

在小步长假设下，非线性约束被一阶化为

```tex
\langle \boldsymbol{w}, \boldsymbol{\varphi}\rangle=0
```

也就是方向必须位于球面在当前点的切空间中。于是求下降方向的问题转为“把梯度投影到切空间后归一化”。

## 典型步骤

1. 从源文的最速下降目标出发，明确损失下降目标和更新步长约束。
2. 写出参数可行域约束，例如 `||w-eta phi||_2=1` 和 `||w||_2=1`。
3. 使用小步长一阶近似，把非线性可行域约束化为线性正交条件。
4. 引入待定系数，将梯度方向修正为满足切空间条件的方向。
5. 得到投影方向，例如球面情形下的 `g-<g,w>w`。
6. 如果一阶更新只近似保持约束，使用回缩把参数拉回约束面。

## 直觉

约束面上的可行更新不能随意离开当前曲面。切空间给出了“在一阶近似下仍留在曲面上”的方向集合；把梯度投影到切空间，就是在所有一阶可行方向中保留真正能下降的分量，丢掉会把参数推出约束面的分量。

## 边界

- 小步长一阶近似是方法成立的基本前提；大步长时需要更严格的回缩或重新求解。
- 球面上的投影公式不能直接搬到所有约束；Stiefel、谱球面等情形要重新写出各自的切空间条件或对偶目标。
- 几何图像可以帮助理解，但源文提醒代数求解过程才是更可靠的主体；跨约束迁移时应优先检查方程结构，而不是只看几何相似。
- 如果约束不是光滑等式约束，或当前点无法给出清楚的一阶切空间，该方法不能直接套用。

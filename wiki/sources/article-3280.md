# article-3280: 高斯型积分的微扰展开（三）

## 文章核心问题

通过重新参数化（改变小参数）来改进高斯型积分微扰展开的收敛性质，并寻找最优参数变换。

## 主要结论

1. 以 λ = (√(a²+6ε)-a)/2 为新小参数展开，得到效果更好的渐近级数。
2. 一般二次参数变换 ε = b₂λ²+b₁λ 可以系统化：通过优化 b₁,b₂ 使高阶展开更简单。
3. 特别地，取 b₁=2a, b₂=13 可使 a₂=0，得到精确到二阶的封闭近似 √(13π/(10a+3√(a²+13ε)))。
4. 该封闭近似在所有测试情形下均优于第一篇的近似(4)式。

## 推导结构

- 用 λ = (√(a²+6ε)-a)/2 作小参数，代入积分并在指数中展开
- 推广到一般二次变换 ε = b₂λ² + b₁λ
- 令 a₂=0 寻找最优参数：b₁=2a, b₂=13
- 得封闭近似 √(13π/(10a+3√(a²+13ε)))

## 关键公式

- λ-展开：∫exp(-ax²-εx⁴)dx = √(π/(a + λ - 7λ²/(6a) + 47λ³/(6a²) - ...))
- 优化封闭近似：∫exp(-ax²-εx⁴)dx ≈ √(13π/(10a+3√(a²+13ε)))

## 体现的方法

- **参数变换微扰展开** → 通过重参数化改变小参数的定义，优化展开的收敛性质

# article-1929: 轻微的扰动——摄动法简介(3)

## 文章核心问题

探讨摄动法在解微分方程中的应用，重点介绍直接级数展开法和小参数展开法，以及摄动法中出现的长期项（久期项）问题。

## 主要结论

1. 微分方程的解可以展开为自变量t的泰勒级数，但收敛性和计算量是限制因素。
2. 摄动法将解展开为小参数ε的幂级数，将非线性微分方程转化为无穷多道线性微分方程。
3. 长期项 t sin t 的出现是摄动法解周期性问题的核心困难。
4. 为消除长期项，发展了PL(Poincaré–Lindstedt)方法、多尺度方法、PLK方法、平均值方法等。

## 推导结构

- E.g.1: dx/dt = sin x + t, x(0)=0 → 泰勒级数展开
- E.g.2: x'' + x = εx², x(0)=1, x'(0)=0 → 摄动展开
  - 设 x = x₀(t) + εx₁(t) + ε²x₂(t) + ...
  - ε=0时: x₀''+x₀=0 → x₀(t)=Cos t
  - O(ε): x₁''+x₁-x₀²=0 → x₁(t)=(3-2Cos t-Cos 2t)/6
  - O(ε²): x₂''+x₂-2x₀x₁=0 → x₂(t)=(-48+29Cos t+16Cos 2t+3Cos 3t+60t Sin t)/144

## 关键公式

- x''+x=εx² 的摄动解：x(t)=Cos t + (ε/6)(3-2Cos t-Cos 2t) + (ε²/144)(-48+29Cos t+16Cos 2t+3Cos 3t+60t Sin t)+...
- 长期项：60t Sin t / 144 → 随时间增长而发散

## 体现的方法

- **摄动级数展开** → 将微分方程解展开为小参数的幂级数，化非线性为无穷线性序列
- **直接泰勒展开** → 将解展开为自变量的幂级数，逐阶求导确定系数

# article-2096: 数学基本技艺之23、24（下）

## 文章核心问题

从23题的解中提炼出一类拟齐次微分方程 dy/dx = Ax^m + Bx^{2m+1}/y 的通解形式的一般规律。

## 主要结论

1. 方程 dy/dx = Ax^m + Bx^{2m+1}/y 有两个特解 y=c₁x^{m+1} 和 y=c₂x^{m+1}，其中 c₁,c₂ 是代数方程 (m+1)c² = Ac+B 的两个根。
2. 通解可写为 (y-c₁x^{m+1})^{kc₁}·(y-c₂x^{m+1})^{-kc₂} = C。
3. 这一规律通过直接对猜想求导验证成立。
4. 体现了"创造"的学习态度：从具体到一般，亲自经历完整推理。

## 推导结构

- 观察23题解的形式 (y+x²/2)(y-x²)² = C
- 提出推广猜想：对 dy/dx = Ax^m + Bx^{2m+1}/y，通解为 (y-c₁x^{m+1})^{α}·(y-c₂x^{m+1})^{β} = C
- 将猜想代入微分方程验证，得到 α:β = c₁:-c₂ 的关系
- 确认猜想的自洽性

## 关键公式

- 推广方程：dy/dx = Ax^m + Bx^{2m+1}/y
- 特解系数：c₁,c₂ 满足 (m+1)c² = Ac+B
- 通解： (y-c₁x^{m+1})^{kc₁}·(y-c₂x^{m+1})^{-kc₂} = C

## 体现的方法

- **拟齐次变量代换法** → 发现并证明一类拟齐次方程的通解结构
- **模式归纳推广** → 从特解到一般形式猜想，再验证自洽性

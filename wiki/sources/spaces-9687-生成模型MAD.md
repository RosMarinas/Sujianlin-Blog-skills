# 当生成模型肆虐：互联网将有"疯牛病"之忧？

- **URL**: https://spaces.ac.cn/archives/9687
- **Author**: 苏剑林
- **Date**: 2023-07-14
- **Category**: 信息时代
- **Tags**: 生成模型, MAD, 模型自噬, 多样性

## Summary

探讨生成模型大规模"肆虐"互联网可能导致的后果——模型自噬紊乱（MAD），即生成模型反复用自己生成的数据迭代训练会导致多样性丧失和质量崩溃。

## Key Concepts

- Model Autophagy Disorder (MAD)：模型自噬紊乱
- 正态分布迭代训练的理论分析：方差衰减、均值偏移
- 截断技巧加速多样性丧失
- 只有持续引入新鲜真实数据才能避免退化

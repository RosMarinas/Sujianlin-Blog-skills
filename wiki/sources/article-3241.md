# article-3241: 高斯型积分的微扰展开（二）

## 文章核心问题

将第一篇中的变系数逼近思想推广到逐阶可计算的形式：将系数A本身展开为ε的幂级数 A = a + a₁ε + a₂ε² + ...，通过逐阶消去被积函数的差异来得到高阶近似。

## 主要结论

1. 将 A 展开为 A = a + a₁ε + a₂ε² + ...，比较被积函数的逐阶差异，令各阶为0。
2. 解得 a₁=3/(2a), a₂=-39/(8a³), a₃=657/(16a⁵)...
3. 得到渐近展开 ∫exp(-ax²-εx⁴)dx = √(π/(a + 3ε/(2a) - 39ε²/(8a³) + 657ε³/(16a⁵) - ...))。
4. 相比直接展开(3)式效果更好，但仍为渐近级数，在 a→0 时发散。
5. 渐近级数虽然收敛半径为0，但截断前几项可在一定范围内获得不错近似。

## 推导结构

- 设 A = a + a₁ε + a₂ε² + a₃ε³ + ...
- 考虑 ∫[exp(-ax²-εx⁴) - exp(-Ax²)] dx = 0
- 被积函数按ε展开，逐项积分
- 令各阶系数为0，解方程组得 a₁, a₂, a₃

## 关键公式

- ∫exp(-ax²-εx⁴)dx = √(π/(a + 3ε/(2a) - 39ε²/(8a³) + 657ε³/(16a⁵) - ...))

## 体现的方法

- **指数变系数逐阶展开** → 将指数中的系数展开为ε的幂级数，各阶分别消去

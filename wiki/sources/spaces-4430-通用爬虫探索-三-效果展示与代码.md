# 通用爬虫探索（三）：效果展示与代码

## 内容概要
文章展示了通用无监督爬虫在博客系统、Discuz 论坛、phpBB 论坛及天涯论坛的实际爬取和楼层切分效果。并给出了用 Python 2.7 编写的完整爬取框架与论坛提取代码。

## 关键内容
1. **爬取效果验证**：
   - 证明了方案能普适于一般博客网站（图1）。
   - 对主流开源论坛（Discuz, phpBB）有很好的解析能力（图2、图3）。
   - 对企业自研的复杂论坛（天涯论坛）同样具有良好的分块切分效果（图4）。
2. **改进空间与局限性分析**：
   - *取舍限制*：为了速度和工程实用性，使用中文占比代替了较精确的语言模型，存在一定误删。
   - *层次丢失*：基于模板扣减的方法提取出的文本是一维的，丢失了原始的嵌套和位置层次信息，难以完全恢复。
   - *缺失视觉信息*：由于缺乏视觉渲染特征（如坐标、大小等），无法像人眼一样直接判定某些属性（如用户名）。
3. **Python 2.7 代码实现**：
   - 给出了 `crawler` 类的具体实现，包括标准 URL 请求、lxml5 解析、DFS DOM 树遍历（包含例外标签处理）、局部差分峰值（peak）查找、以及中英文/日期过滤打分逻辑。
   - 给出了用于论坛帖子和回帖属性抽取合并的 `extract_info` 函数实现。

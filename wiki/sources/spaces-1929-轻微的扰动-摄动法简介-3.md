## 文章核心问题
探讨微分方程摄动解法的两种展开思路（自变量泰勒级数展开 vs 小参数级数展开），引入摄动级数解中出现的“长期项/久期项”这一根本缺陷，并介绍数学物理中消除长期项的典型策略。

## 主要结论
自变量直接展开法随着求导阶数增加计算量会崩溃且收敛区间窄。摄动法将解展开为小参数 $
arepsilon$ 的幂级数，能够把原方程的优良性质保留下来，但其核心困难在于极易产生长期项（如 $t\sin t$）。这些项在物理上随时间增长而发散，破坏了解的周期性。为消除长期项，数学家提出了 PL方法、多尺度方法、PLK方法和平均值方法。

## 推导结构
1. **直接泰勒级数展开 (E.g.1)**：针对 $
rac{dx}{dt} = \sin x + t, x(0)=0$，通过逐次求导求出 $x(0), x'(0), x''(0)$ 等值，写出时间自变量 $t$ 的幂级数解。指出其收敛性无法保证且高阶项求导困难。
2. **小参数摄动展开 (E.g.2)**：针对非线性振子方程 $x'' + x = 
arepsilon x^2$（带初值），假设解 $x(t) = x_0(t) + 
arepsilon x_1(t) + \dots$，代回方程分解出嵌套的线性方程级数。
3. **求解线性级数**：解出 $x_0(t)=\cos t$, $x_1(t)=
rac{1}{6}(3-2\cos t-\cos 2t)$，及带有长期项 $60t\sin t$ 的 $x_2(t)$。
4. **引入长期项及消除法**：说明长期项产生机制，并给出 PL、多尺度、PLK、平均值等进阶方法的引言。

## 关键公式
- 非线性振子方程: $x''+x = 
arepsilon x^2$
- 带有长期项的二阶解: $x_2(t) = 
rac{1}{144}(-48 + 29\cos t + 16\cos 2t + 3\cos 3t + 60t\sin t)$

## 体现的方法
- `[[摄动级数展开法]]`：将非线性微分方程线性化为无穷多道线性微分方程的分步求解法。

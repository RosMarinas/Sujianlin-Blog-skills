# 借助变分法变换坐标

- **URL**: https://spaces.ac.cn/archives/3181
- **Author**: 苏剑林
- **Date**: 2015-01-06
- **Category**: 数学研究
- **Tags**: 变分法, 坐标变换, 拉普拉斯算子

## Summary

介绍通过变分法来简化ODE和PDE的坐标变换过程。通过对作用量进行坐标变换再变分，比直接代入计算更为简洁。

## Key Concepts

- 通过作用量变分简化ODE坐标变换（如二体问题）
- 通过作用量变分简化PDE坐标变换（如拉普拉斯算子变换到极坐标、球坐标）
- 正交坐标系下的雅可比行列式与作用量变换

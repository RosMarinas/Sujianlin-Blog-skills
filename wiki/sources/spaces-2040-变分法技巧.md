# 变分法的一个技巧及其"误用"

- **URL**: https://spaces.ac.cn/archives/2040
- **Author**: 苏剑林
- **Date**: 2013-07-30
- **Category**: 数学研究
- **Tags**: 变分法, 欧拉-拉格朗日方程, 短程线

## Summary

介绍变分法中处理根号形式被积函数的技巧（平方再变分），以及该技巧可能被误用的场景。核心技术是在变分过程中将ds视为参数。

## Key Concepts

- 根号形式的被积函数的变分简化技巧
- 短程线和相对论作用量中的变分
- 技巧的适用边界：必须对整个拉格朗日函数平方，而非部分

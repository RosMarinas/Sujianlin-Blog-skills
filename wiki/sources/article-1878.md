# article-1878: 轻微的扰动——摄动法简介(1)

## 文章核心问题

介绍摄动法（微扰理论）的基本思想：先忽略小量求解精确解，再将解表达为小参数的幂级数逐步修正。

## 主要结论

1. 摄动法的核心步骤：令小参数ε=0求得精确解，再将解表达为ε的幂级数。
2. 通过代数方程 εx³ + x² = p² 的求解直观展示摄动法的操作流程。
3. 摄动法可应用于代数方程、微分方程等各种领域。
4. 在量子力学中，摄动法和变分法是最基本的两种近似方法。

## 推导结构

- 以代数方程 εx³ + x² = p² 为例：
  - 令 ε=0 得 x₀ = ±p
  - 设 x = p + a₁ε + a₂ε² + ...
  - 代入展开，逐阶匹配系数
  - 解得 a₁ = -p²/2, a₂ = 5p³/8
  - 近似解 x ≈ p - (p²/2)ε + (5p³/8)ε²

## 关键公式

- εx³ + x² = p² 的摄动解：x = p - (p²/2)ε + (5p³/8)ε²
- 数值验证：ε=0.1, p=1 时近似解 0.95625，精确解 0.9554

## 体现的方法

- **摄动级数展开** → 将解表示为小参数ε的幂级数，逐阶求解

# 《新理解矩阵2》：矩阵是什么？

## 文章核心问题
从几何视角解释矩阵和矩阵乘法的本质，说明其作为坐标系及坐标变换的直观物理意义，并简要介绍张量的概念。

## 主要结论
1. 矩阵 $A$ 的列向量是仿射坐标系的基向量，运算 $y=Ax$ 描述了新坐标系 $A$ 下的向量 $x$ 在直角坐标系 $I$ 下的坐标度量。
2. 矩阵作用于向量有两种等价诠释：其一，点不变，坐标系发生变换；其二，直角坐标系中的点被“运动”变换到新位置。
3. 矩阵乘法 $AB$ 代表将矩阵 $B$ 描述的坐标系（其分量基于坐标系 $A$）变换为基于直角坐标系 $I$ 的测量结果。
4. 数字、向量、矩阵可分别对应 0 阶、1 阶、2 阶张量，三维立方阵对应 3 阶张量。

## 推导结构
1. 通过矩阵列向量的几何含义，指出矩阵代表一个平面仿射坐标系。
2. 通过 $Ax=y$ 的具体二阶矩阵数值算例，说明新旧坐标系坐标转换的几何一致性。
3. 阐释“运动是相对的”：点变换与坐标系变换的对偶性。
4. 解释矩阵乘矩阵 $AB$ 的几何意义以及维度一致性的要求。
5. 顺承向量与矩阵的构成方式，引出高维立方阵——张量的概念。

## 关键公式
- 仿射坐标变换：$Ax = y$ (其中 $A$ 的列向量为基， $x$ 为在新基下的坐标， $y$ 为直角坐标系下的坐标)
- 矩阵乘法几何测量：$AB$

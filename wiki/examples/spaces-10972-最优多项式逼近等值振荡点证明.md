# 最优多项式逼近等值振荡点证明

该反证法是多项式插值与逼近论中极其著名的切比雪夫定理（Chebyshev's Equioscillation Theorem）证明的构造细节。

对于区间 $[a,b]$ 上的连续函数 $g(x)$，其不超过 $n$ 阶的最佳逼近多项式 $f^*(x)$ 在无穷范数下满足：
$$
f^* = \mathop{\text{argmin}}_f \max_{x\in[a,b]} |f(x) - g(x)|
$$
等值振荡定理指出，$f^*$ 为最佳逼近的充要条件是，存在至少 $n+2$ 个交替等值振荡点 $a\leq x_0 < x_1 < \cdots < x_{n+1} \leq b$，使得误差在此处交替达到最大值：
$$
f^*(x_k) - g(x_k) = (-1)^{k+\sigma} \max_{x\in[a,b]} |f^*(x) - g(x)|
$$
其中 $\sigma\in\{0,1\}$。

定理必要性部分的证明基于无穷范数方向导数与Kolmogorov定理。具体证明通过严密的代数反证法进行：

1. **假设反证**：假设交错等值振荡点的最大组数为 $m$，且 $m \le n+1$。
2. **寻找分割点**：在这些符号交替的最大误差区域之间，必然存在 $m-1$ 个交替符号的分割点 $z_1 < z_2 < \dots < z_{m-1}$，它们将不同符号的极值区域完全隔开。
3. **构造方向多项式**：利用这些分割点作为根，我们主动构造一个特殊的子空间方向多项式 $h(x) = \pm(x - z_1)(x - z_2)\cdots(x - z_{m-1})$。因为 $m-1 \le n$，所以 $h(x)$ 的最高次数不超过 $n$，属于给定的逼近多项式函数子空间。
4. **推导最优性矛盾**：无穷范数 $\|f-g\|_\infty$ 的最优化要求，对任意多项式修正方向 $u(x)$，其单侧方向导数 $\mathcal{D}_x[\max|f-g|, u] = \lim_{\epsilon\to 0^+} \frac{\|f+\epsilon u - g\|_\infty - \|f-g\|_\infty}{\epsilon}$ 在最优点处必须非负。然而，通过恰当选择 $h(x)$ 前面的全局符号 $\pm$，可以使得在所有取得最大误差的集合点上，多项式 $h(x)$ 的符号与误差函数 $f^*(x) - g(x)$ 的符号完全相反。这意味着如果让 $f^*$ 沿着 $h(x)$ 的方向微小移动，所有产生最大误差处的绝对值都会同时严格下降，从而导致沿着 $h(x)$ 修正时的无穷范数方向导数严格为负。这与 Kolmogorov 导数必须非负的最优性条件产生了不可调和的代数矛盾。

因此，最初的假设不成立，最佳多项式逼近必然交替地穿过原函数，且交替等值振荡点的数目 $m$ 必须满足 $m \ge n+2$。

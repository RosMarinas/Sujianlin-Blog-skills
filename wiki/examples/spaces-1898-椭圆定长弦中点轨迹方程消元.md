# 椭圆定长弦中点轨迹方程消元

本例详细记录了椭圆内滑动定长弦中点的高次曲线轨迹方程的代数推导过程。该解法由网友“理想”提出，其核心思想是通过构造和差变量并利用无量纲化进行代数消元。

首先，设椭圆长、短轴分别为 $2a$、$2b$，定弦长为 $2r$，弦两端点为 $A(x_1, y_1)$ 和 $B(x_2, y_2)$，中点为 $P(x, y)$。定义半弦在坐标轴上的投影偏差分别为 $w = \frac{x_1 - x_2}{2}$ 和 $h = \frac{y_1 - y_2}{2}$。由于 $x = \frac{x_1 + x_2}{2}$ 和 $y = \frac{y_1 + y_2}{2}$，可利用端点所在椭圆的性质建立坐标间的约束。

基于上述变量，将点 A 和 B 的椭圆方程相加与相减，并结合弦长条件 $|AB|=2r$，可以建立三个关键的代数方程：
1. 相加方程（由两端点椭圆方程相加变形而得）：$\frac{x^2+w^2}{a^2} + \frac{y^2+h^2}{b^2} = 1$
2. 相减方程（由两端点椭圆方程相减并平方，用以消去负号）：$\frac{x^2}{a^2}\cdot\frac{w^2}{a^2} = \frac{y^2}{b^2}\cdot\frac{h^2}{b^2}$
3. 弦长方程（由两点间距离公式而得）：$w^2 + h^2 = r^2$

为了消去未知偏差变量 $w^2$ 和 $h^2$，得到仅含中点坐标 $x, y$ 的方程，求解中引入了无量纲量 $p=\frac{x}{a}, q=\frac{y}{b}$ 以及 $m=\frac{w}{a}, n=\frac{h}{b}$。将上述三方程分别转化为：
- $p^2 + q^2 + m^2 + n^2 = 1$
- $p^2m^2 = q^2n^2$
- $a^2m^2 + b^2n^2 = r^2$

通过联立这三个方程：首先用弦长方程分别乘以 $p^2$ 和 $q^2$，代入相减方程化简，可得到关系式 $(a^2q^2 + b^2p^2)(m^2+n^2) = (p^2+q^2)r^2$。接着，利用第一个无量纲方程得出 $m^2+n^2 = 1 - p^2 - q^2$ 并代入上述关系，即可彻底消去弦差项 $m$ 与 $n$。将 $p, q$ 替换回原坐标变量 $x, y$，最终化简得到的高次轨迹方程为：

$$
(\frac{x^2}{a^2} + \frac{y^2}{b^2} - 1)(\frac{x^2}{a^4} + \frac{y^2}{b^4} + \frac{r^2}{a^2b^2}) + \frac{r^2}{a^2b^2} = 0
$$

# spaces-4195: Viterbi词典硬解码干预

本例展示了如何将传统离散词典先验（通过概率加权）注入深度学习序列标注网络的解码阶段，实现神经网络泛化能力与人工硬干预的代数融合。

在基于字标注法（通常使用 `SBME` 标签，分别代表单字、词首、词中、词尾）的神经网络中文分词模型（如基于全卷积网络 FCN 的模型）中，模型会输出输入文本中每个字符位置对应的标签对数概率矩阵（记为 `nodes`）。然而，纯神经网络模型可能会在特定领域的专有名词上发生切分错误。为了修正这些错误，我们可以引入用户自定义的干预词典，并通过硬解码的方式在 Viterbi 算法执行前直接对发射概率进行数值干预。

具体硬解码干预的实现步骤如下：

1. **加载干预词典与预计算权重**：首先读取用户提供的外部词频词典，将干预词汇及其对应权重转换为对数形式保存：`add_dict[word] = np.log(weight)`，该对数得分记为权重分 `f`。
2. **文本区间匹配**：在待切分的原始输入文本中，利用多模式匹配或正则表达式精确查找干预词典中的词汇，获取每个匹配到的词汇在文本中的起止区间 `[i.start(), i.end()]`。
3. **代数偏置字级发射概率矩阵**：
   - 如果匹配到的干预词是单字（长度 $L = 1$），则直接在该单字对应的字符位置上，对其 `S`（单字）标签的对数概率累加相应的权重得分：`nodes[i.start()]['s'] += f`。
   - 如果匹配到的干预词汇长度 $L \ge 2$，则需要对该区间内多个字符位置的标签概率进行结构化累加：
     - 在其首字位置累加 `B`（词首）标签的概率：`nodes[i.start()]['b'] += f`；
     - 在其尾字位置累加 `E`（词尾）标签的概率：`nodes[i.end()-1]['e'] += f`；
     - 对于该词区间内的所有中间字位置 $j \in (i.start(), i.end()-1)$，一律累加 `M`（词中）标签的概率：`nodes[j]['m'] += f`。
4. **Viterbi 全局动态规划寻优**：经过上述局部偏置修改后，我们将被人工干预修正后的发射概率矩阵 `nodes` 输入到 Viterbi 算法中求解全局最优路径。由于干预词汇对应的标准切分标签的对数概率被 `f` 显著拉高，Viterbi 算法在动态规划回溯时，会强制或强烈倾向于选择包含这些词汇完整切分的最优状态序列。

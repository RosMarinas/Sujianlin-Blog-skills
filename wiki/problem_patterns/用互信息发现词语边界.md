# 用互信息发现词语边界

## 四步算法

1. 统计单字频率和相邻字共现频率
2. 对满足 $p_{ab} < min\_prob$ 或 $PMI < min\_pmi$ 的邻字进行切分
3. 统计切分后的"准词语"频率，保留高频词
4. 去冗：删除可用子词概率乘积表示的候选词

## 推广

该模式也可用于更一般的"基本元素合并"问题，如词组合并为词组（window=1时的句模版算法）。

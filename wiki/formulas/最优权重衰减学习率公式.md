# 最优Weight Decay和Learning Rate调度公式


## 概述

（待补充）

## 记忆周期
s <= -log c / (lambda*eta)

## 常数WD最优LR调度
eta_s ~ eta_max / (lambda*eta_max*s + 1)

## 联合最优调度(lambda_s = alpha*eta_s)
eta_s ~ eta_max / sqrt(2*lambda_max*eta_max*s + 1)
lambda_s ~ lambda_max / sqrt(2*lambda_max*eta_max*s + 1)

# 常系数线性常微分方程算子分解公式

## 概述

对于一般的 $n$ 阶常系数线性常微分方程：
$$(D^n + a_1 D^{n-1} + \dots + a_n)y = f(x)$$

其特征多项式 $x^n + a_1 x^{n-1} + \dots + a_n = 0$ 的特征根为 $r_1, \dots, r_n$。算子可以因式分解为：
$$(D - r_n)\dots(D - r_1)y = f(x)$$

通解公式即为嵌套的一阶逆算子的叠加：
$$y = (D - r_n)^{-1}\dots(D - r_1)^{-1} f(x)$$

通过代入一阶逆算子公式并化简，即得出多重积分嵌套形式。在重根时（如 $r_{n-1}=r_n$），公式依然成立并自动在积分里产生 $x$ 的多项式，达到了齐次与非齐次、重根与非重根解法的统一。

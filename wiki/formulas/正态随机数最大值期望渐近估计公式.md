# 正态随机数最大值期望渐近估计公式


## 概述

（待补充）

## 公式

```tex
\mathbb{E}[z_{\max}] \sim \sqrt{2\log \frac{n}{\sqrt{2\pi}}}
```

## 符号与条件

$z_1,\dots,z_n$ 是独立标准正态随机变量，$z_{\max}=\max\{z_1,\dots,z_n\}$，$n$ 是样本个数。条件是 $n\to\infty$ 的渐近估计；源文同时给出主阶 $\sqrt{2\log n}$，并说明带 $\sqrt{2\pi}$ 修正的表达式在数值上更准。

## 用途

该公式用于估算大量正态样本的最大值量级。源文给出三条证据路径：Jensen 不等式可快速得到 $\mathbb{E}[z_{\max}]\le \sqrt{2\log n}$；最大值分布 $P(z_{\max}\le z)=[\Phi(z)]^n$ 结合 Laplace 近似给出更精细主点；逆变采样也导向同一渐近结果。应用上，源文用它估计低精度 Attention logits 中重复最大值的概率。

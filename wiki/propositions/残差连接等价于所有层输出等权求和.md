# 残差连接等价于所有层输出等权求和

## 内容

将残差连接x_t = x_{t-1} + f_t(x_{t-1})改写为y的视角：设y_t = f_t(x_{t-1})，则x_t = y_0 + y_1 + ... + y_t，于是y_{t+1} = f_{t+1}(y_0 + ... + y_t)。这说明残差连接本质上是对所有层输出做等权求和。

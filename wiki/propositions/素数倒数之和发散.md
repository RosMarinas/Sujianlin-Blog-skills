proof_route: |
  1. 引入敛散性一致引理：对于正项数列 $a_i > 0$，其级数和 $S = \sum a_i$ 与乘积项 $T = \prod (1+a_i)$ 满足：
     $$1+S < T < e^S$$
     由此两者的敛散性充分必要。
  2. 考虑截断素数 $p$ 的欧拉乘积：
     $$\prod_{q \le p} \frac{q}{q-1} = 1+\frac{1}{2}+\frac{1}{3}+\dots > \ln(p+1)$$
  3. 利用 $\frac{p_n}{p_n-1} < 1+\frac{1}{p_{n-1}}$ 进行放缩：
     $$\ln(p+1) < \prod_{q \le p} \frac{q}{q-1} < 2\prod_{q \le p} (1+\frac{1}{q})$$
  4. 应用引理 $T < e^S$ 放缩乘积项：
     $$\ln(p+1) < 2 e^{Q} \quad \text{其中 } Q = \sum_{q \le p} \frac{1}{q}$$
  5. 对不等式两边取自然对数，整理得到双对数下界估计：
     $$Q > \ln \ln (p+1) - \ln 2$$
  6. 由于存在无穷多个质数且随着 $p \to \infty$，下界 $\ln \ln (p+1) \to \infty$，故全体素数倒数之和发散。

# 素数倒数之和发散

素数倒数之和发散是数论中一个比“素数无穷”更强的定理。由于自然数倒数和（调和级数）发散，而平方倒数和收敛，素数倒数和的发散表明素数在自然数中的分布并没有想象中那么稀疏。欧拉对这一发散性的证明也是解析数论中不等式放缩与连续逼近的重要示范。

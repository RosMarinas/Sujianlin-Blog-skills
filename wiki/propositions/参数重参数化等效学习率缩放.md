title: 参数重参数化等效学习率缩放
statement: '在神经网络权重参数做尺度重参数化 $\boldsymbol{\theta} = \lambda \boldsymbol{\phi}$ （$\lambda$ 为常数比例）时，优化变换后的参数 $\boldsymbol{\phi}$ 数学上完全等价于对原参数 $\boldsymbol{\theta}$ 的学习率进行调整。对于常规 SGD 优化器，其等效学习率变为 $\lambda^2 \alpha$；对于自适应学习率优化器（如 Adam, RMSprop），其等效学习率变为 $\lambda \alpha$。'

proof_route: |
  1. 考虑原参数的梯度下降公式：$\boldsymbol{\theta}_{n+1} = \boldsymbol{\theta}_n - \alpha \frac{\partial L(\boldsymbol{\theta}_n)}{\partial \boldsymbol{\theta}_n}$。
  2. 引入尺度重参数化 $\boldsymbol{\theta} = \lambda \boldsymbol{\phi}$。根据多元复合函数求导链式法则，关于新参数 $\boldsymbol{\phi}$ 的梯度为：
     $\frac{\partial L(\lambda \boldsymbol{\phi}_n)}{\partial \boldsymbol{\phi}_n} = \frac{\partial L(\boldsymbol{\theta}_n)}{\partial \boldsymbol{\theta}_n} \frac{\partial \boldsymbol{\theta}_n}{\partial \boldsymbol{\phi}_n} = \lambda \frac{\partial L(\boldsymbol{\theta}_n)}{\partial \boldsymbol{\theta}_n}$。
  3. 对于 SGD，新参数 $\boldsymbol{\phi}$ 的更新步骤为：
     $\boldsymbol{\phi}_{n+1} = \boldsymbol{\phi}_n - \alpha \frac{\partial L(\lambda \boldsymbol{\phi}_n)}{\partial \boldsymbol{\phi}_n} = \boldsymbol{\phi}_n - \lambda \alpha \frac{\partial L(\boldsymbol{\theta}_n)}{\partial \boldsymbol{\theta}_n}$。
     两边同时乘以常数 $\lambda$，依据 $\boldsymbol{\theta} = \lambda \boldsymbol{\phi}$ 恢复为原参数表达：
     $\boldsymbol{\theta}_{n+1} = \boldsymbol{\theta}_n - \lambda^2 \alpha \frac{\partial L(\boldsymbol{\theta}_n)}{\partial \boldsymbol{\theta}_n}$。
     对比可知，等效于将原先的学习率 $\alpha$ 变成了 $\lambda^2 \alpha$。
  4. 对于 Adam 等自适应优化器，其更新量分母包含关于梯度的二阶矩平方根。
     由于梯度 $\boldsymbol{g}_{\boldsymbol{\phi}} = \lambda \boldsymbol{g}_{\boldsymbol{\theta}}$。
     一阶动量估计 $\boldsymbol{m}_{\boldsymbol{\phi}} = \lambda \boldsymbol{m}_{\boldsymbol{\theta}}$；
     二阶矩估计 $\boldsymbol{v}_{\boldsymbol{\phi}} = \lambda^2 \boldsymbol{v}_{\boldsymbol{\theta}}$，其开方项 $\sqrt{\boldsymbol{v}_{\boldsymbol{\phi}}} = \lambda \sqrt{\boldsymbol{v}_{\boldsymbol{\theta}}}$。
     因此自适应更新量为：
     $\boldsymbol{u}_{\boldsymbol{\phi}} = \frac{\boldsymbol{m}_{\boldsymbol{\phi}}}{\sqrt{\boldsymbol{v}_{\boldsymbol{\phi}}}} = \frac{\lambda \boldsymbol{m}_{\boldsymbol{\theta}}}{\lambda \sqrt{\boldsymbol{v}_{\boldsymbol{\theta}}}} = \frac{\boldsymbol{m}_{\boldsymbol{\theta}}}{\sqrt{\boldsymbol{v}_{\boldsymbol{\theta}}}}$。
     新参数更新公式为 $\boldsymbol{\phi}_{n+1} = \boldsymbol{\phi}_n - \alpha \boldsymbol{u}_{\boldsymbol{\phi}}$。两边同乘 $\lambda$ 恢复原参数形式：
     $\boldsymbol{\theta}_{n+1} = \boldsymbol{\theta}_n - \lambda \alpha \frac{\boldsymbol{m}_{\boldsymbol{\theta}}}{\sqrt{\boldsymbol{v}_{\boldsymbol{\theta}}}}$。
     可见自适应优化器下的等效学习率缩放仅为 $\lambda$。

## 工程意义

该命题建立了“参数尺度缩放”与“学习率调整”之间的数学等价桥梁。

在深度学习框架（如 Keras）中，修改优化器底层的逐层参数更新规则非常繁琐（常需要继承 `Optimizer` 并重写核心计算图）。而本命题提供了一种优雅的“事前”等效方案：即只需通过自定义层或 Wrapper 在前向计算时将参数乘以 $\lambda$，并在初始化时将权重对应除以 $\lambda$，即可在使用现有的、未经修改的通用优化器（如 Adam）时，无缝达到调整该层学习率的工程目的。

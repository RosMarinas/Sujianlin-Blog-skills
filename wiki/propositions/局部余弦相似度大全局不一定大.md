proof_route: |
  1. 写出两个子空间拼接下的全局余弦表达式：
     \cos(\boldsymbol{x},\boldsymbol{y}) = \frac{\cos(\boldsymbol{x}_1, \boldsymbol{y}_1) \Vert\boldsymbol{x}_1\Vert \Vert\boldsymbol{y}_1\Vert+ \cos(\boldsymbol{x}_2, \boldsymbol{y}_2)\Vert\boldsymbol{x}_2\Vert \Vert\boldsymbol{y}_2\Vert}{\sqrt{\Vert\boldsymbol{x}_1\Vert^2 + \Vert\boldsymbol{x}_2\Vert^2} \sqrt{\Vert\boldsymbol{y}_1\Vert^2 + \Vert\boldsymbol{y}_2\Vert^2}}
  2. 设定局部余弦满足 \cos(\boldsymbol{x}_1, \boldsymbol{y}_1) = c_1 \ge \lambda，\cos(\boldsymbol{x}_2, \boldsymbol{y}_2) = c_2 \ge \lambda。
  3. 通过参数控制，令 \Vert\boldsymbol{x}_1\Vert = \epsilon 且 \Vert\boldsymbol{y}_2\Vert = \epsilon，而 \Vert\boldsymbol{x}_2\Vert = 1 且 \Vert\boldsymbol{y}_1\Vert = 1。
  4. 代入全局余弦展开式，分子为：
     c_1 \epsilon + c_2 \epsilon = (c_1 + c_2)\epsilon
     分母为：
     \sqrt{1 + \epsilon^2}\sqrt{1 + \epsilon^2} = 1 + \epsilon^2
  5. 令 \epsilon \to 0，则分子趋于 0，分母趋于 1。因此全局余弦相似度极限值为：
     \lim_{\epsilon \to 0} \cos(\boldsymbol{x}, \boldsymbol{y}) = 0
     这证明了无法利用正的局部下界 \lambda 提供大于 0 的全局下界。
  6. 证明上界：假设不失一般性有 c_1 \le c_2，利用柯西不等式可得：
     \cos(\boldsymbol{x},\boldsymbol{y}) \le c_2 \left[\frac{\Vert\boldsymbol{x}_1\Vert \Vert\boldsymbol{y}_1\Vert + \Vert\boldsymbol{x}_2\Vert \Vert\boldsymbol{y}_2\Vert}{\sqrt{\Vert\boldsymbol{x}_1\Vert^2 + \Vert\boldsymbol{x}_2\Vert^2}\sqrt{\Vert\boldsymbol{y}_1\Vert^2 + \Vert\boldsymbol{y}_2\Vert^2}}\right]
     方括号部分是二维模长向量的夹角余弦，必然小于或等于 1。因此有：
     \cos(\boldsymbol{x},\boldsymbol{y}) \le c_2 = \max(c_1, c_2)
     上界定理得证。

# 局部余弦相似度大全局不一定大

## 命题陈述

在高维向量拼接或直和的几何结构中，局部每个维度块的余弦夹角方向一致性（局部余弦 $\ge \lambda > 0$）不能为全局整体余弦相似度提供非零的下界。全局余弦可以任意趋近于零，且其上界受到局部最大余弦相似度的限制。

## 物理意义

这一命题在机器学习模型（如大模型）参数分析中具有核心的警示价值：在局部统计指标（如逐层计算参数梯度一致性夹角）表现优异的情形下，在整体参数空间（如 70 亿参数的全局联合空间）中计算的整体一致性可能由于层间更新幅度的分配极度不均而退化到零。这强调了在大模型特征分析中，尺度效应和范数权值分配的重要性。

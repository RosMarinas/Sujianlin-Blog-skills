title: 常数权重衰减下最优学习率解
statement: 在常数权重衰减参数 lambda 下，为使神经网络在预训练过程中各步数据对模型的贡献度（记忆系数）保持均等，最优的学习率调度轨迹随着训练步数 s 的推进服从 s^-1 反比演化：eta_s = eta_max / (lambda * eta_max * s + 1)。
assumptions:
  - 自适应齐次梯度更新器的平均场近似成立
  - 动量因子 beta_1, beta_2 近似为 0
  - 要求所有训练步的加权记忆系数相等

proof_route: 在指数滑动平均（EMA）视角下，权重为 theta_t = (1-lambda*eta)*theta_t-1 - eta*u_t。在动态学习率 eta_i 下，迭代展开后第 j 步梯度的加权累积系数正比于 e^kappa_j * eta_j，其中 kappa_j = sum_{i=1}^j eta_i * lambda_i。令其恒等于常数，可得差分控制方程 e^(lambda*eta_j)*eta_j = eta_j-1。用导数近似差分，得到微分方程 lambda * eta_s + dot_eta_s / eta_s = 0。求解该一阶常微分方程，初值 eta_0 = eta_max，即得 eta_s approx eta_max / (lambda * eta_max * s + 1)。

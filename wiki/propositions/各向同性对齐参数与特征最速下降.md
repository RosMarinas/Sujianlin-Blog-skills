proof_route: 1. 考虑 SGD：参数更新方向为 -eta * X^T * dL/dY。将其乘回特征，可得特征层面的变化量 Delta Y = -eta * X * X^T * dL/dY。由特征空间的最速下降定义，特征的最优更新方向应与 -dL/dY 对齐。若且仅若 X X^T = c I 时，Delta Y = -eta * c * dL/dY，方向一致。 2. 考虑 Muon：更新量为 -eta * msign(X^T * dL/dY)。特征变化为 Delta Y = -eta * X * msign(X^T * dL/dY)。若 X X^T = c I，代入矩阵符号函数解析展开，化简可得 Delta Y approx -eta * msign(dL/dY)，特征层面上同时以最大谱范数最速下降，方向和幅值完美匹配。

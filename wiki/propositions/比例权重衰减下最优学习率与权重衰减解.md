title: 比例权重衰减下最优学习率与权重衰减解
statement: 当权重衰减与学习率满足正比例更新条件 lambda_s = alpha * eta_s 时，要使每步训练样本的记忆系数均等，最优的学习率与权重衰减调度序列必须按训练步数的平方根倒数进行同步衰减：eta_s, lambda_s \propto (2 * alpha * eta_max^2 * s + 1)^-0.5。

  - 正比例约束关系 lambda_s = alpha * eta_s 成立
  - 自适应齐次型优化器的平均场近似成立
  - 样本记忆系数在优化轨迹上恒为常数

proof_route: 将比例更新条件 lambda_s = alpha * eta_s 代入均等记忆微分方程 lambda_s * eta_s + dot_eta_s / eta_s = 0，得到非线性微分方程 alpha * eta_s^2 + dot_eta_s / eta_s = 0。整理为 dot_eta_s / eta_s^3 = -alpha。对其积分，应用初值条件 eta_0 = eta_max，可求得解析解 eta_s = eta_max / sqrt(2 * alpha * eta_max^2 * s + 1)。由比例条件同理可得 lambda_s 形式。

该比例命题解决了常数衰减下 LR 降为 0 导致权重坍缩的问题，在数学上证明了“LR-WD 同步以平方根速率进行衰减”是最优调度格式。

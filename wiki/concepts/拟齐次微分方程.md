# 拟齐次微分方程

## 定义

拟齐次微分方程是一类具有特殊齐次性的非线性一阶微分方程，其形式为 dy/dx = Ax^m + Bx^{2m+1}/y。这类方程的特点是：
- 存在 y=cx^{m+1} 形式的特解
- 可通过变量代换 x = t·y^k 化为可分离变量形式
- 通解具有 (y-c₁x^{m+1})^{α}(y-c₂x^{m+1})^{β}=C 的乘积结构

## 通解规律

对于方程 dy/dx = Ax^m + Bx^{2m+1}/y：
- c₁, c₂ 是 (m+1)c² = Ac+B 的两个根
- 通解：(y-c₁x^{m+1})^{kc₁}(y-c₂x^{m+1})^{-kc₂} = C

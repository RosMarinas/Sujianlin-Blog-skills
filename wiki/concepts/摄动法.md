# 摄动法

## 定义

摄动法（Perturbation Method）是一种系统化的近似求解方法，广泛应用于代数方程、微分方程和物理系统的分析。其核心思想是：如果问题中包含一个小参数ε，先忽略它（令ε=0）得到一个可精确求解的问题，然后将真实解表达为ε的幂级数，逐阶确定系数。

## 关键步骤

1. 识别小参数ε
2. 令ε=0，求解无摄动问题（基本解）
3. 假设解为 x = x₀ + εx₁ + ε²x₂ + ...
4. 代入原方程，按ε的幂次展开
5. 令各阶系数为零，得到递推的线性方程序列
6. 逐阶求解

## 应用场景

- 代数方程：三次方程 εx³ + x² = p²
- 微分方程：x'' + x = εx²（非线性转化为无穷线性序列）
- 量子力学：几乎所有无法精确求解的问题都依赖摄动法

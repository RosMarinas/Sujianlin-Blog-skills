# 熵不变性

## Definition

熵不变性是指Attention分布a_{i,j}的熵H_i应该对序列长度n不敏感。即增加新token时，已有token的注意力分布不应被过度分摊。

## Key Formula

H_i = -∑a_{i,j} log a_{i,j} ≈ log n - 0.24λd + O(1)
为抵消n的影响，令log n - 0.24λd = 0，得λ = κ log n / d

## 应用

熵不变性导出新的缩放因子λ = κ log n / d，其中κ = sqrt(d)/log 512时在n=512退化为标准缩放因子。此设计在MLM任务中表现出更好的长度外推能力。

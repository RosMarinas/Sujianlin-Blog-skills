# 捷径模型 (Shortcut Model)

**捷径模型**（Shortcut Model, arXiv:2410.12557）是2024年底提出的扩散模型加速方法，核心思想是将生成步长ε作为条件输入到速度模型中，配合自一致性损失函数，实现单阶段（从零）训练即可单步生成。ICLR 2025全8分接收。

Shortcut模型建立在ReFlow ODE框架之上。设$\boldsymbol{x}_t = (1-t)\boldsymbol{x}_0 + t\boldsymbol{x}_1$（$\boldsymbol{x}_0$为噪声，$\boldsymbol{x}_1$为数据），标准ReFlow训练速度模型$\boldsymbol{v}_\theta(\boldsymbol{x}_t, t)$逼近$\boldsymbol{x}_1 - \boldsymbol{x}_0$。Shortcut的将步长ε作为额外条件$\boldsymbol{v}_\theta(\boldsymbol{x}_t, t, \epsilon)$，使模型能自适应不同离散化步长。

损失函数由两部分组成：**精确ODE条件**$\mathcal{L}_1$（训练$\epsilon=0$时的模型使其逼近$\boldsymbol{x}_1 - \boldsymbol{x}_0$）和**自一致性正则**$\mathcal{L}_2$（训练使$2\epsilon$步长走一步等于$\epsilon$步长走两步）。$\epsilon$取9个离散值$\{0, 2^{-7}, ..., 1\}$，通过嵌入加入模型。训练时混合batch：3/4样本训练$\mathcal{L}_1$，1/4训练$\mathcal{L}_2$，并对$\mathcal{L}_2$教师项使用stop_gradient。

Shortcut模型与AMED（中值定理加速）共享核心洞见：要实现极低NFE甚至单步生成，必须修改模型本身而非求解器。在MeanFlow（文章30）的视角下，Shortcut的自一致性条件是平均速度性质的直接体现。

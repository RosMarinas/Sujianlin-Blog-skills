# 滑动平均视角下的权重衰减和学习率

## 概述
将Weight Decay写成滑动平均形式，推导最优的WD Schedule和LR Schedule。核心是"每个Batch同等重要"假设下反解最优调度。

## 核心思想
1. Weight Decay使参数做滑动平均，记忆周期约O(1/lambda*eta)。
2. 每个Batch同等重要假设可反解最优调度。
3. 自适应学习率优化器共享类似结论。

## 关键公式
记忆周期: s <= -log c / lambda*eta
常数WD最优LR: eta_s ~ eta_max / (lambda*eta_max*s + 1)
联合调度: eta_s ~ eta_max / sqrt(2*lambda_max*eta_max*s + 1)

## 假设
- 平均场近似在Update RMS、Weight RMS估计中表现合理
- 对Adam、RMSProp、SignSGDM、Muon等齐次优化器均适用

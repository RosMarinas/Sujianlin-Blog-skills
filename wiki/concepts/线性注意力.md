# 线性注意力

## Definition

线性注意力（Linear Attention）是标准Softmax Attention的线性化变体，通过去除softmax中的指数归一化，将QK^T矩阵的二次复杂度降为线性复杂度O(nd^2)。其核心形式为O = (QK^T ⊙ M)V，在Causal场景下可写为线性RNN形式。

## Key Properties

1. Causal形式可递归计算：o_t = S_t q_t, S_t = S_{t-1} + v_t k_t^T
2. 复杂度关于序列长度线性：O(nd^2)
3. 本质上是一个线性RNN（cumsum或带遗忘门的加权求和）
4. 天然低秩：Attention矩阵的秩不超过head_size d
5. 适合Decoder-only架构的推理加速

## Variants

- 原始线性Attention（cumsum）
- 带遗忘门（RetNet风格）
- DeltaNet（基于Delta Rule）
- Gated DeltaNet（带遗忘门的Delta Rule）
- MesaNet（基于解析解）
